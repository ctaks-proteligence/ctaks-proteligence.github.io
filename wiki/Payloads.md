# Payloads

# Purpose

The purpose of Payloads is to provide the layout information for Payload
data.

# Usage

## Common

These are fields or sets of fields referenced by the base message
classes or by specific messages.

## Message Classes

These are the base message formats upon which all specific messages are
based.

## Specific Messages

These are messages of a specific type (e.g., ADJUST) and subtype (e.g.,
Q). For each specific message, the type value and base class for the
message's Message and messageDetail definitions are shown. The Message
extended class shows the basic class used for the specific message while
the messageDetail extended class shows the base messageDetail contents
for the specific message. If additional tags are also listed, those
fields are in addition to the base message detail fields.
